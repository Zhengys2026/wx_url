# wx_url

wx_url 是一个微信和企业微信对接授权获取 用户昵称 头像 open_id ，可以动态设定其url转发携带参数。

</div>

## 💡 wx_url

`1` 平台授权配置
   - [x] 提供简易模式，无需操作编排
   - [ ] 插件封装功能

`2` URL使用配置
   - [x] 多库复用，混用
   - [ ] 支持文件阅读器
   - [ ] 更多的数据预处理方案
  
  
`3` 运营能力
   - [x] 免登录分享窗口
